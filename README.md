
# Backend Microservices

This project is a **backend microservices application** built using **Spring Boot** and **Spring Cloud**.

It shows how multiple backend services can work together in a microservices architecture.

---

## Technologies Used

* Java
* Spring Boot
* Spring Cloud
* Eureka Server
* Spring Cloud Config
* API Gateway

---

## Services in This Project

* **Config Server** – stores configuration in GitHub
* **Eureka Server** – service discovery
* **API Gateway** – routes requests
* **Backend Services** – handle business logic

Each service runs independently.

---

## How to Run

1. Start **Config Server**
2. Start **Eureka Server**
3. Start **API Gateway**
4. Start other backend services

---

## Project Purpose

* Learn microservices
* Understand Spring Cloud
* Practice backend development
